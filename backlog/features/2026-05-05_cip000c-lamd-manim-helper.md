# Task: Create lamd_manim_helper.py

## Description

Create `lamd/util/lamd_manim_helper.py` — the source for the generated `_lamd_manim.py` helper module. This is Phase 1 of CIP-000C.

This module is generated by `mdpp.py` at build time (written alongside the `.slides.manim.py` output) and provides text conversion utilities so that generated Manim scene code remains clean and readable.

## Acceptance Criteria

- [ ] `_md_to_pango(text)` converts `**bold**` → `<b>bold</b>` and `*italic*` → `<i>italic</i>`
- [ ] `lamd_text(md_string, font_size=36)` splits on `$...$`, renders math segments as `MathTex` and text segments as `MarkupText`, returns a `VGroup` arranged horizontally (or a single Mobject if only one part)
- [ ] `lamd_display_math(latex_string, font_size=48)` returns a centred `MathTex` for display/block equations
- [ ] All functions handle empty string input without error
- [ ] Unit tests in `lamd/tests/test_lamd_manim_helper.py` cover: plain text, bold, italic, inline math, mixed text+math, empty string, display math

## Implementation Notes

```python
import re
from manim import *

def _md_to_pango(text):
    text = re.sub(r'\*\*(.+?)\*\*', r'<b>\1</b>', text)
    text = re.sub(r'\*(.+?)\*', r'<i>\1</i>', text)
    return text

def lamd_text(md_string, font_size=36, **kwargs):
    parts = re.split(r'(\$[^$]+\$)', md_string)
    mobjects = []
    for part in parts:
        if part.startswith('$') and part.endswith('$'):
            mobjects.append(MathTex(part[1:-1], font_size=font_size, **kwargs))
        elif part.strip():
            mobjects.append(MarkupText(_md_to_pango(part), font_size=font_size, **kwargs))
    if not mobjects:
        return VGroup()
    return mobjects[0] if len(mobjects) == 1 else VGroup(*mobjects).arrange(RIGHT)

def lamd_display_math(latex_string, font_size=48, **kwargs):
    return MathTex(latex_string, font_size=font_size, **kwargs).move_to(ORIGIN)
```

Unit tests can mock the Manim imports using `unittest.mock` to avoid requiring a full Manim install for unit test runs.
