Writing Guidelines

  1. Content Style

    \ifndef{clearName}
\define{clearName}

\section{Clear Title}

\notes{
Start with overview...
Provide details...
End with summary...
}

\endif

    • Comment complex macros

    • Provide examples

Common Patterns

  1. Content Structure

\ifndef{topic}
\define{topic}

\section{Topic}

% Context-specific content
\ifdef{SLIDES}
    \slides{...}
\else
    \notes{...}
\endif

\endif

  1. Media Handling

\figure{\includediagram{name}{width}}
{Clear caption}{fig:label}

\code{
# Clear, commented code

}
