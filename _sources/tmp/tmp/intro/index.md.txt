# Getting Started with LaMD

LaMD is a macro system for creating academic content that can be rendered in multiple formats from a single source. This section will help you get started.

```{toctree}
:maxdepth: 2

installation
quickstart
```

Key benefits of LaMD:

* Write once, use in multiple formats (slides, notes, papers)
* Maintain consistency across materials
* Reuse content efficiently
* Separate content from presentation
